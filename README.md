### 1- What is CI/CD?

**CI/CD** stands for **Continuous Integration** and **Continuous Deployment/Delivery**.
It is a set of practices that automate the process of building, testing, and deploying code, enabling faster and more reliable software delivery.

---

### 2- CI/CD Tools

Some popular CI/CD tools include:

* **Jenkins**
* **GitHub Actions**
* **GitLab CI/CD**

---

### 3- What is Jenkins Pipeline?

A **Jenkins Pipeline** is a collection of plugins that allows you to define and automate the software development lifecycle (build, test, deploy) as code using a pipeline script.

---

### 4- Jenkins Programming Language

Jenkins is primarily written in:

* **Java**

---

### 5- Pipeline Scripting Language

Jenkins Pipeline syntax is based on:

* **Groovy**

---

### 6- Types of Jenkins Pipelines

You can write pipelines in two main ways:

1. **Declarative Pipeline**

   * Simple and structured syntax
   * Easier to read and maintain

2. **Scripted Pipeline**

   * More flexible
   * Written entirely in Groovy
